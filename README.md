# AURA Macro

AURA Macro is a lightweight Python macro application built specifically with **macOS** in mind. It allows users to automate repetitive keyboard and mouse actions using customizable macros and global hotkeys. The project is designed to be simple, fast, and easy to expand as new features are added.

## Features

* macOS-friendly design
* Global hotkey support
* Start and stop macros instantly
* Keyboard automation using PyAutoGUI
* Responsive interface built with Tkinter
* Threaded macro execution to keep the UI responsive
* Expandable macro system for future recording and playback

## Built With

* Python
* Tkinter
* PyAutoGUI
* Pynput
* Threading

## Current Status

This project is actively being developed. The current version includes a working macro engine with hotkey support and a simple graphical interface.

## Why AURA Macro?

Many macro tools focus primarily on Windows. AURA Macro is designed with **macOS users** in mind, providing a simple and responsive automation tool that feels at home on Apple's desktop platform while remaining easy to learn and customize.
